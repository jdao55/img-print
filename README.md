# Dependencies
- c++17 compiler
- CMake > 3.15
- Conan

# Building
1. clone repository 
```
git clone https://github.com/jdao55/img-print.git && cd img-print
```
2. Create build directory and Run Cmake
```
mkdir build && cd build
cmake ../ 
```
3. Compile program
```
make -j4
```
